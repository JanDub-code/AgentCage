AgentCage

ac — Run AI coding agents in a disposable rootless Podman container.

Mounts only the current directory, keeps your home and credentials out of reach, and auto-deletes the container when the agent exits. One command, one repo, no traces.
linux fedora-first rootless podman
GitHub README
Quick start
$ ac # build/init the local runtime image $ ac codex # run Codex CLI $ ac claude # run Claude Code $ ac opencode # run Opencode $ ac antigravity # run Google Antigravity CLI $ ac shell # open a contained shell
First ac builds the local runtime image. Each agent downloads lazily on first run.
